# DroidHaz v1.1.0

見下ろし型サバイバル・アクションシューター（ウェーブ制ローグライト）。静的PWA＋Capacitor（Android）。

## 同梱 APK について
- `droidhaz-debug.apk` は **未署名のデバッグビルド**（動作確認・サイドロード用）です。
- インストール手順: Android 端末で「不明なアプリのインストール」を許可してから APK を開いてください。
- Play ストア等への正式配布には署名付き release ビルドが必要です（手順は `CAPACITOR.md`）。

## v1.1.0 の新要素
- **プレイアブルキャラクター5種**: 主人公・魔法使い・女子高生・骸骨・ロボット。タイトル画面で選択。
  各キャラ専用の武器がドロップ・使用可能（魔法／投擲／骨・呪い／エネルギー兵装など）。
- **被ダメージ等の4モーション**: 被弾・弱体化・瀕死・息切れを表情と姿勢で表現。
- **敵攻撃バリエーション**: 電撃（感電スタン）・毒（継続ダメージ）・炎上（継続ダメージ）と、
  それぞれ専用のダメージモーション。新敵: 帯電体・胞子虫・火炎甲虫。
- **精緻な手続き描画スプライト**（表情・服が分かる拡大表示）。

## 安定性・品質（v1.0.0 からの監査対応）
- Capacitor ネイティブ機能の復活（Android 戻るボタン／自動ポーズ／ハプティクス／セーブ永続化）。
- ウェーブ進行・タッチスティック・PWA(Service Worker)・描画パフォーマンス・self-XSS などの修正。
- ユニットテスト **199件**（`node --test`）、全モジュール構文チェック、CI で自動実行。

## 操作
移動: WASD/矢印・画面スティック ／ 射撃: K ／ 近接: J ／ リロード: R ／ 武器切替: 1-5 ／ ダッシュ: Shift ／ ポーズ: Esc。
スマホ/タブレットでは画面上ツインスティックが自動表示されます。

## 既知の制約
- 本リリースの APK は未署名デバッグビルド（prerelease）。署名付き配布は keystore secret 設定後にタグ push で生成。
- StatusBar の色制御は Android 15/16+ の edge-to-edge で限定的。
