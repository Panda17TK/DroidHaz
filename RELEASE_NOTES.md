# DroidHaz v1.2.2

見下ろし型サバイバル・アクションシューター（ウェーブ制ローグライト）。静的PWA＋Capacitor（Android）。

## 同梱 APK について
- `droidhaz-debug.apk` は **未署名のデバッグビルド**（動作確認・サイドロード用）です。
- インストール手順: Android 端末で「不明なアプリのインストール」を許可してから APK を開いてください。
- Play ストア等への正式配布には署名付き release ビルドが必要です（手順は `CAPACITOR.md`）。

## v1.2.2 の修正（見た目の調整）
- **顔のドット崩れを修正**: ピクセルアートのドット密度を上げ（DOT=4）、輪郭・陰影を緩和。実ゲームサイズで目・口がくっきり判読可能に。
- **斜め/横向きの顔を修正**: 顔パーツのはみ出しと「宙に浮く鼻」を解消。横向きでも奥目を小さく残し、自然な向きに。
- **女子高生**: 優しげな眉を追加し、頬の赤みを削除。

## これまでの要素（v1.0〜1.2.1）
- プレイアブル **10体**: 既存5体（主人公・魔法使い・女子高生・骸骨・ロボット）＋新ジョブ5体
  **アコライト・アサシン・ウィッチ・ソードマスター・マーシナリー**。全員キャラ専用武器（ドロップ解放あり）。
- **8方向スプライト**＋**ピクセルアート化（ドット格子）**。武器は照準方向に追従、上向きで後ろ姿。
- 敵攻撃バリエーション（電撃／毒／炎上）と専用被ダメージモーション。被弾・弱体化・瀕死・息切れの4モーション。
- ダメージ／息切れボイスの吹き出し、ステージ制＋エンドレス、強化カード、中ボス／ボス、キルカム。
- Capacitor ネイティブ統合（戻るボタン／自動ポーズ／ハプティクス／Preferences永続化）、PWA オフライン。

## 操作
移動: WASD/矢印・画面スティック ／ 射撃: K ／ 近接: J ／ リロード: R ／ 武器切替: 1-5 ／ ダッシュ: Shift ／ ポーズ: Esc。
スマホ/タブレットでは画面上ツインスティックが自動表示されます。

## 品質
- ユニットテスト **219件**（`node --test`）、全モジュール構文チェック、CI で自動実行。

## 既知の制約
- 本リリースの APK は未署名デバッグビルド（prerelease）。署名付き配布は keystore secret 設定後にタグ push で生成。
- StatusBar の色制御は Android 15/16+ の edge-to-edge で限定的。
